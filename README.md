# gPPTX Generator

Welcome to **gPPTX**, an AI-powered tool that turns your notes into PowerPoint presentations instantly.

## Features
*   **Multiple Inputs**: Reads from README.md, .txt, and .yaml files.
*   **AI Integration**: Uses powerful AI (Ollama/Gemma) to summarize and structure your content.
*   **Custom Design**: Generates slides with a bold, red-themed design.
*   **Images**: Optionally include an image in your slides.

## How to Use
1.  Start the application (see `INSTALL.txt` for setup).
2.  Click **"Select Input Files"** to choose your text documents.
3.  (Optional) Click **"Select Image"** to add a picture to your slides.
4.  Click **"Generate Presentation"**.
5.  Choose where to save the file.
6.  Wait for the "Done!" message.
